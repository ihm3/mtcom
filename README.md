## mtcom

mtcom is a simple command-line serial communication tool for Windows, written in PowerShell.
It provides a lightweight, Linux-like serial port experience on Windows systems.

## Requirements
- Windows
- PowerShell 5.1 or later

## Usage
Run the script from PowerShell by specifying the serial port and baud rate.

Example:
```powershell
.\main.ps1 -Port COM3 -BaudRate 9600
```
Adjust the port and communication settings as needed.
